使用Github Page + Hexo搭建静态个人博客

前言

​ 很多时候你以为你理解了某样东西，但其实并没有。将自己的理解写成文章，就是一个自我检查的过程，当你尝试把一个东西说清楚时，你才知道你实际上对这个东西还有哪些问题。写成博客的形式，不仅可以帮助其他人，简历加分，更重要的是可以让别人指出你没有意识到的错误，从而改进。

大致流程

​ 使用Hexo框架将Markdown笔记解析成静态网页，再使用插件将网页push到特定的git仓库，由GitHub Pages发布网站。

​ Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

​ GitHub Pages 是一项静态站点托管服务，它直接从 GitHub 上的仓库获取 HTML、CSS 和 JavaScript 文件，（可选）通过构建过程运行文件，然后发布网站。
